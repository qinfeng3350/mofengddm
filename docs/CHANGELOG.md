# 更新日志

本文档记录墨枫低代码平台（Portal 前端与相关能力）的主要变更。版本号与发布节奏以团队实际发版为准；此处按功能主题归纳近期更新。

---

## 未发版 · 近期迭代

### 帮助与文档

- 首页头部提供「帮助中心」入口（图标下拉），包含：使用文档、开发文档、更新日志、接口文档。
- 上述文档以站内页面展示 Markdown 内容，便于随版本迭代维护。

### 开发者页面

- 将原「仪表盘」入口调整为「开发者页面」，支持在应用内编写、保存自定义页面代码（本地存储），用于后续对接自定义报表或填写界面。
- 「新建页面」入口与开发者页面路由：`/app/:appId/developer`。

### 表单数据列表

- 打开表单数据列表时，视图下拉默认始终为「全部数据」，不会自动选中新创建的自定义视图。
- 新建自定义视图后，仍保持「全部数据」为当前选中项。
- 自定义视图支持在视图下拉中通过删除图标移除，并带确认；删除当前正在使用的视图时会回到「全部数据」。
- 数字字段与公式字段在列表中统一展示千位分隔符（如 `7,999`），便于阅读金额与统计类数据。

### 表单设计器

- 移除「列表设计」页签（该能力未就绪，避免误导）。

### 子表与公式

- 子表内公式按行计算（如 `{单价} * {数量}`），支持在子表列中引用同行列字段；主表公式可聚合子表列（如 `SUM({采购明细.总价})`）。
- 公式引擎支持常见全角符号自动归一（括号、逗号、运算符等），减少因输入法全角符号导致的解析失败。
- 修复子表公式在渲染阶段回写表单导致的 React 报错：`Cannot update a component while rendering a different component`；公式写回移至 `useEffect` 中批量处理。
- 修复新增子表行后公式不计算或主表合计未更新的问题（含默认值为公式模式 `formulaEdit` 的列）。

### 子表交互

- 在只读、非编辑等不可编辑状态下，子表「新增」「导入」「删除」按钮禁用（灰色不可点），避免误操作。

### 其他

- 首页文档入口由多个平铺链接收拢为「帮助中心」下拉，界面更简洁。
